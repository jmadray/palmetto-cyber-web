# Palmetto Cyber Defense Website

Professional website for Palmetto Cyber Defense LLC — a cybersecurity MSP focused on small and medium businesses.

## Quick Start

This is a static site designed for GitHub Pages. To deploy:

1. Create a new repo (e.g., `palmetto-cyber-web` or `palmettocyber.github.io`)
2. Push these files to the `main` branch
3. Go to Settings → Pages → Source: Deploy from branch (`main`, `/root`)
4. Your site will be live at `https://yourusername.github.io/repo-name/`

## Structure

```
├── index.html    # Main page
├── styles.css    # All styling
└── README.md     # This file
```

## Customization

- **Contact email**: Update the email link in `index.html`
- **Colors**: Edit CSS variables in `:root` in `styles.css`
- **Content**: Edit sections directly in `index.html`
